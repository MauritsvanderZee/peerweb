In this tutorial you will learn how you can handle your time account for the jong ondernemen. It will show how to create several tasks, manage them, make an overview over your time consumption, move your time to another task and store this inside your time account.



Open the Peerweb Homepage


Open your preferred web browser and use this link www.fontysvenlo.org/peerweb . While loading the webpage the browser will ask you about accepting a certificate. Continue with a click on [ Accept ] . After that the login screen appears in the web browser.

Enter in the text field “peer_id” your student id number and in the “password” field your password. Click [ login ] to enter the site.

If you forgot your password you can request it with the fields “logincode” ( your student id number ) and “Date of Birth” ( your birthday in the DD-MM-YYYY format) on the lower right corner of the login site. If you press [ Submit ] an e-mail with the password is generated and sent to your default E-mail address (this is usually your fontys e-mail address ).




The Time Management System




After you’ve entered the peerweb you may click on the tab which is labeled with “Time and presence” (1). Now a new page will be opened. On the left side a menu will show up. It contains the following topics:





Help


This topic is still under construction.



Presence ( not necessary for jong ondernemen )


Behind this topic you will find a listing of your presence at every hour per day. Every time you login to peerweb your presence is stored. The fontys logo will show your presence in this hour and a red cross will show your absence for this time.



Group Presence


This topic contains a listing of all presence in every educational hour from every jong ondernemen member. In the drop-down menu you can choose your project ( in your case MINI : MINI ondernemen 2005 ), the day and the hour you want to know. Click the [ absend ] button to update the page. The fontys logo will show the presence of this member for this hour and a red cross will show the absence of this member.



Group Time


In this topic the given time of every member will be shown. In the drop-down menu you can choose the project of interest. Additionally your whole time you’ve invested in this project will show up in the drop-down menu.

Under this drop-down menu the time of every project member will be displayed. This information will be divided on the several tasks every member has made.



Project tasks:


Under project tasks you can create several tasks for a particular project. In the ‘Project’ drop-down menu you can choose your project, for which you want to create a new task. In the text field which is labeled with ‘Task description’ you can add a short description of the task. With the ‘new’ button you can create a new task, with the ‘update’ button you can update the description of a task.


WARNING:

In the current version it is possible to create and update tasks with an empty task description. It has no affects on the system but it can cause some confusion.

WARNING:

In the current version you can’t delete a task when it is created once so be careful and do not create useless tasks.


Time book keeping


Behind this topic you can update and reallocate your time consumption to other tasks. It is possible to cut your time consumption from one task and add it to another task. For reallocating the time of a particular task just click on the scissor behind the task and a new window will opened.



Set up your Time Management System


The time management system is not active when you enter the “Time and presence” topic for the first time. It must be started manually the first time. Before you can start it for the first time, a task must be created where the time can be appended.



Go to “Project Tasks” and select under “Project” ( 1 ) ‘PERSON : Personal timekeeper project’. Write into “Task description” (2) ‘idle’ and finally click on [ new ] ( 3 ). Now you’ve created a new task.

This task has the purpose of saving the time which are consuming for you you need for sleeping, eating and so on. In other words it contains anything that has nothing to do with the jong undernemen.

Now create a second task but select now under “Project” ‘MINI : Mini ondernemingen 2005’. For the task description you can use any description that describes your work proper.

Now you can create a task for every work that you are doing for jong undernemen.


WARNING:

In the current version it is possible to create and update tasks with an empty task description. It has no affects on the system but it can cause some confusion.

WARNING:

In the current version you can’t delete a task when it is created once so be careful and do not create useless tasks.


After creating all necessary tasks you can activate the time management system with a click on the clock icon ( 4 ). Now a new part of the page will be displayed below the tabs. With this part you can start a task.



In the drop-down menu ( 1 ) you can select the current task and with a click on the start-button (2) the task will be started. Behind the start-button the current time of this task will be shown. For changing the task just select another task ( 1 ) and click the start-button ( 2 ).

If you want to stop all tasks change the task to “PERSON:idle” ( 1 ) and press the start-button ( 2 ) .

Button 3 is currently without any function, just ignore it.


INFORMATION:

It is not necessary that you stay all the time online. Only for changing tasks and creating tasks it is required to be logged into peerweb. All the other time you can be offline. The time of the last selected task will still count up until you change it to another task.



Split time up to different tasks


Eventually it can happen that you forget to change a task. But there is still a way to correct this. Open in the menu on the left side the topic “Time book keeping”. Under this topic all your consumed time will be available. On the right hand of every topic a scissor icon will be displayed. If you need to refine the time amount of a particular task click on its scissor icon and a new page will be opened.



In the drop-down menus 2 and 3 you may select your target task on which the split time should be added up. In textfield 1 you can select the point of time, when the task should be splitted up. This time must lie between the values, which are given at the position A and B. Finally cut the task with a click on [ try ] ( 4 ). Now close the window with a click on [ close ].

The “Time book keeping” page be updated now.
