This page allows you to create a subversion repository for a project.

The created repository contains a directory per group that is readable and writable by the group and readable by all tutors. The names of the repositories is derived from the group aliases. For rights management and configuration of the repository the subversion subdirs conf and hooks are also versioned and read/write for the tutors. This allows you as a tutor to update group rights in the repository.
