Deep copy a project/milestone

Copy the selected project-milestone with all its attributes, settings and members into a project/milestone.
This is particularly handy if you start a project with the same set of students. Note that all is copied from the source project, including tutors, group names and roles.
The base data of the target project/milestone must exists.

Select target project/milestone from the list of available empty ones and

The list of target projects is the set of project-milestones that have no members yet and are owned by you.
If this list is empty, then you can either create a new project or add a milestone to an existing project.
