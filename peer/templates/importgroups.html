Import groups

This page allows you to upload predefined student group memberships into peerweb. You can use it to put students in already defined groups. To use this page, some preparation on your side is in order.
  • Make sure you have (as in: are owner) a project and
  • You have the defined groups that you want to use already defined in the milestone you want to use for the new student groups
  • Make sure all students are available (present) in peerweb.
Furthermore, you need to prepare the student to group information in a specific way:
  • It is a xlsx (excel >= 2007) file.
  • The file should contain one worksheet called 'Worksheet'. any other worksheets will be ignored. See the example in the screen shot on the right.
  • The first row should contain the column names of the data to import.
  • of these column names ony snummerand grp_num will be used. All other column names will be ignored.
  • From row 2, the first column you should contain the student number, digits only, typically 7 of them.
  • The second column should contain the intended group number typically starting at 1 and ending at the number of groups defined for the peerweb project this import is for. Digits only.
  • The name of the file is irrelevant. It will be removed from the server after use.
When all preparations are in order, select the target project+milestone in the form on the left, select the file to upload in the upload widget and press upload.
